# CodeOrigins Documentation

Welcome to the CodeOrigins documentation. Here you can find information about how this project is built and functions.

## Features
- **3D Visualization:** Uses `react-three-fiber` and `drei` to render a timeline of programming languages.
- **Data-Driven:** The languages data is loaded dynamically supporting i18n via separate JSON files for each locale.
- **Timeline & Filters:** Explore the languages by adjusting time bounds and applying filters on documentation, usage, or categories.
